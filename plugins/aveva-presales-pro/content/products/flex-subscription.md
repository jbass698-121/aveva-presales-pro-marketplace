# AVEVA Flex Subscription Positioning

> **Note:** This is a starter file covering the Flex subscription commercial model. Distributors should overwrite with current Flex terms, regional adjustments, and customer talk tracks.

## What Flex Is

AVEVA Flex is a subscription-based commercial model that lets customers access the full AVEVA software portfolio through a unified credits-based subscription rather than per-product perpetual licenses. The model emphasizes financial agility, scalability, and continuous access to product updates.

## Value Proposition

**Financial agility:**
- Reduced upfront commitment vs. perpetual license model.
- CapEx → OpEx shift, easier budgetary fit for many customers.
- Ability to scale up or down based on operational needs.

**Technical advantages:**
- Continuous access to updates, including new modules and capabilities.
- Cloud-enabled deployment with seamless integration to AVEVA Connect.
- Bridge between digital and physical operations through digital twin strategies.

**Business outcomes:**
- Faster time-to-value through proactive support included.
- Workforce flexibility — credits can be reallocated as priorities shift.
- Predictable ongoing cost vs. lumpy CapEx renewal cycles.

## Key Talking Points by Buyer Persona

**For the CFO / Finance leader:**
- Predictable annual OpEx instead of multi-year CapEx commitments.
- Credits model means you're not paying for unused product entitlements.
- Easy alignment with corporate budget cycles.

**For the operations director:**
- Access to the full portfolio means you can deploy the right tool for the right problem without re-procuring.
- Updates flow continuously; you're never stuck on an old version.

**For the IT / OT director:**
- Cloud-enabled deployment removes the patch-management overhead.
- Single subscription simplifies vendor management.

## Common Objections

### "We prefer perpetual licenses."
**Response:** "Perpetual makes sense when the deployment is static and the buyer has CapEx headroom. For most operators today, the operational reality is: you're modernizing in phases, you want continuous access to updates, and you want CapEx flexibility. Flex matches that operational rhythm. We can also offer a hybrid path — start with perpetual on the core SCADA, add Flex credits for advanced modules — if the financial structure helps."

### "Subscription costs more long-term."
**Response:** "When you compute fully-loaded TCO including support, upgrades, and obsolescence risk, perpetual is rarely cheaper at year 5. Subscription gets you to ROI faster because you're using current features from day one, not waiting for the next CapEx cycle to upgrade."

### "We need on-premises deployment."
**Response:** "Flex supports both cloud and on-premises deployments. The commercial model is independent of where the software runs. You get subscription benefits regardless of deployment architecture."

## Distributor Considerations

- **Margin model:** verify with AVEVA channel ops the current Flex partner discount tiers.
- **Renewal motion:** subscription renewals are an annual conversation — invest in customer success plays from day 1.
- **Hybrid customers:** some customers run perpetual + Flex side-by-side. Pricing math is non-trivial; involve the AVEVA pricing desk on hybrid deals.
