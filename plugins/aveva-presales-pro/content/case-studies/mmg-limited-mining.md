# Case Study: MMG Limited

## Customer Profile

- **Industry:** Mining (copper, cobalt, lead, zinc)
- **Footprint:** 16 facilities globally including Sepon, Kinsevere, Las Bambas
- **Size:** Mid-tier multinational miner

## The Pain

Production and reliability inconsistency across 16 facilities; mine-to-mill optimization opportunities going uncaptured; capacity below nameplate at multiple operations.

## The Solution

AVEVA Production Management deployed across 16 factories. PI System for time-series intelligence. System Platform for unified operations visibility.

## The Outcome

| Metric | Result |
|---|---|
| Global capacity improvement | 10-20% |
| Sepon mine production | 12% above nameplate |
| Kinsevere mine production | 34% above nameplate |
| Additional copper production | 29,422 tonnes (2015) |

**Headline ROI:** Direct incremental revenue capture from production above nameplate.
